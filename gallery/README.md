# Gallery Instructions

## How to add photos

1. **Go to:** `https://github.com/NatalieC001/Hummingbird-in-Paper/tree/main/gallery`

2. **Click into the folder** where you want to add photos (progress_photos, failures, armature, etc.)

3. **Click "Add file" → "Upload files"**

4. **Drag your photos** from your computer into the box

5. **Before committing**, look at the filename box. Make sure each filename follows this pattern:
folder-name/photo-name.jpg

Example: `armature/jointforge-keys.jpg`

6. **Scroll down and click "Commit changes"**

---

## File naming rules

| Do | Don't |
|----|-------|
| `progress_photos/wing-test-01.jpg` | `wing test 01.jpg` (spaces break it) |
| `failures/snapped-neck.jpg` | `failures/Snapped Neck.jpg` (capitals are fine but spaces bad) |
| `armature/jointforge-plugin-demo.mp4` | `armature/demo.mp4` (too vague) |

**Best practices:**
- Use lowercase letters
- Use dashes `-` or underscores `_` instead of spaces
- Keep names short but descriptive
- Use `.jpg`, `.jpeg`, `.png`, `.gif`, or `.mp4`

---

## Fix broken images

If an image shows only the name (no picture):

1. The file is either:
- Not a supported format (JPG, PNG, GIF, MP4)
- Corrupted
- The path is wrong

2. **Delete the broken one:**
- Click into the folder
- Click the checkbox next to the broken file
- Click "Delete" at the top
- Commit the deletion

3. **Upload it again** following the rules above

## Full path examples

gallery/armature/jointforge-screenshot.png
gallery/progress_photos/week3-body-assembly.jpg
gallery/failures/cracked-wing-closeup.jpg
gallery/armature/jointforge-plugin-demo.mp4



**Incorrect:**
gallery/armature/photo 1.jpg ← has a space
gallery/armature/IMG_5423.JPG ← works but not descriptive
gallery/armature/ ← missing filename
